בפוסט הקודם הצגתי מודולים בתור "זה כמו מרחב וקטורי רק מעל חוג אבל בעצם זה ממש לא כמו מרחב וקטורי כי תשכחו מכל מה שידעתם על מושגים כמו בסיס ופרישה". המקבילה שלנו למרחב שנפרש על ידי וקטורים הייתה תת-מודול שנוצר על ידי איברים; וכאן היו לנו תופעות מוזרות כמו מודול שנוצר על ידי מספר סופי של איברים אבל יש לו תת-מודולים שאין להם קבוצה יוצרת סופית.

בפוסט הזה ננסה, בזהירות, להרים את השברים ולקחת מהם את מה שנזדקק לו בשביל <strong>משפט המבנה של מודולים נוצרים סופית מעל תחום ראשי</strong>.

הדבר הראשון שנעשה יהיה לנסות ולהציל את מושג ה<strong>בסיס</strong>. באלגברה לינארית, קבוצה {% equation %}B{% endequation %} היא בסיס למרחב וקטורי {% equation %}V{% endequation %} מעל שדה {% equation %}F{% endequation %} אם לכל {% equation %}v\in V{% endequation %} קיים צירוף לינארי <strong>יחיד</strong> של אברי הבסיס שנותן אותו. כלומר, קיימים {% equation %}b_{1},\dots,b_{n}\in B{% endequation %} ו-{% equation %}\lambda_{1},\dots,\lambda_{n}\in F{% endequation %} כך ש-{% equation %}v=\sum\lambda_{i}b_{i}{% endequation %}, ודרך הייצוג הזו היא יחידה עד כדי פרמוטציה של האינדקסים של ה-{% equation %}\lambda_{i}b_{i}{% endequation %}. בנוסף, סימנו בתור {% equation %}\dim V{% endequation %} את העוצמה של {% equation %}B{% endequation %} וקראנו לזה <strong>המימד</strong> של {% equation %}V{% endequation %}.

במודולים דבר כזה לא בהכרח קיים, אבל אפשר לדבר על המקרה שבו הוא קיים. אנחנו אומרים ש-{% equation %}R{% endequation %} מודול {% equation %}M{% endequation %} הוא <strong>חופשי</strong> מעל הקבוצה {% equation %}A\subseteq M{% endequation %} אם כל {% equation %}m\in M{% endequation %} ניתן לכתיבה <strong>באופן יחיד</strong> בתור {% equation %}m=\sum_{i=1}^{n}r_{i}a_{i}{% endequation %} עבור {% equation %}r_{1},\dots,r_{n}\in R{% endequation %} ו-{% equation %}a_{1},\dots,a_{n}\in A{% endequation %}. לא כל מודול הוא חופשי, אבל לא קשה להראות שבהינתן חוג {% equation %}R{% endequation %} וקבוצה {% equation %}A{% endequation %} אז <strong>קיים</strong> {% equation %}R{% endequation %}-מודול חופשי מעל {% equation %}A{% endequation %}: מגדירים את {% equation %}M{% endequation %} הזה להיות הקבוצה של כל הביטויים מהצורה {% equation %}\sum_{i=1}^{n}r_{i}a_{i}{% endequation %} (פורמלית: כל הפונקציות {% equation %}f:A\to R{% endequation %} כך ש-{% equation %}f\left(a\right)=0{% endequation %} לכל {% equation %}a\in A{% endequation %} למעט מספר סופי). לא קשה להראות שזה {% equation %}R{% endequation %}-מודול חופשי, ושהוא יחיד עד כדי איזומורפיזם. יותר מכך, הוא מקיים את מה שנקרא <strong>תכונה אוניברסלית</strong> (מושג שטרם הבהרתי עד הסוף, גם בסדרת הפוסטים הזו), שאפשר לתאר בדיאגרמה הקומוטטיבית הבאה:

הדיאגרמה אומרת את הדבר הבא: לכל קבוצה {% equation %}A{% endequation %} קיים {% equation %}R{% endequation %}-מודול {% equation %}F\left(A\right){% endequation %} (זהו המודול החופשי מעל {% equation %}A{% endequation %}) כך שאם {% equation %}M{% endequation %} הוא {% equation %}R{% endequation %}-מודול <strong>כלשהו</strong> וקיימת פונקציה {% equation %}\varphi:A\to M{% endequation %} שמתאימה לכל איבר של {% equation %}A{% endequation %} איבר ב-{% equation %}M{% endequation %}, אז קיים <strong>הומומורפיזם של מודולים</strong> {% equation %}\Phi:F\left(A\right)\to M{% endequation %} כך ש-{% equation %}\varphi\left(a\right)=\Phi\left(a\right){% endequation %} (כאן אנחנו חושבים על {% equation %}a{% endequation %} גם בתור איבר של {% equation %}A{% endequation %} אבל גם בתור איבר של {% equation %}F\left(A\right){% endequation %}; זו המשמעות של החץ מ-{% equation %}A{% endequation %} אל {% equation %}F\left(A\right){% endequation %} שמסמל את העתקת הזהות). כלומר, אנחנו יכולים לחשוב על כל מודול ש"מערב" את {% equation %}A{% endequation %} בתור תמונה של {% equation %}F\left(A\right){% endequation %}; זה המודול ה"כללי ביותר" שמייצג את {% equation %}A{% endequation %} כמודול.

עכשיו אני רוצה לחבר את הדבר הזה לשתי בניות אחרות שאנחנו מכירים מאלגברה לינארית (ומאלגברה בכלל): מכפלות ישרות וסכומים ישרים.

<strong>מכפלה ישרה</strong> של {% equation %}R{% endequation %}-מודולים {% equation %}N_{1},N_{2},\dots,N_{k}{% endequation %} היא הדרך הטבעית להפוך את המכפלה הקרטזית {% equation %}N_{1}\times N_{2}\times\dots\times N_{k}{% endequation %} ל-{% equation %}R{% endequation %}-מודול; מגדירים חיבור וכפל "רכיב רכיב", כלומר {% equation %}\left(a_{1},\dots,a_{k}\right)+\left(b_{1},\dots,b_{k}\right)\triangleq\left(a_{1}+b_{1},\dots,a_{k}+b_{k}\right){% endequation %} ו-{% equation %}r\left(a_{1},\dots,a_{k}\right)\triangleq\left(ra_{1},\dots,ra_{k}\right){% endequation %}. זה אותו רעיון כמו הבניות הדומות שראינו עבור חבורות וחוגים. <strong>סכום ישר</strong> זה אותו דבר בדיוק כמו מכפלה ישרה כל עוד אנחנו מדברים על מכפלה של <strong>מספר סופי</strong> של מודולים, שזה מה שתיארתי למעלה; אם אנחנו לוקחים אינסוף מודולים אז המכפלה הישרה שלהם היא עדיין המכפלה הקרטזית עם חיבור וכפל נקודתיים, אבל סכום ישר לוקח מהמכפלה הקרטזית רק את האיברים שבהם כל הכניסות הן 0 למעט מספר סופי - קצת מזכיר את מה שראינו קודם עם הבניה של המודול החופשי. אני לא הולך לדבר על מכפלות אינסופיות הפעם כי אני לא צריך את זה, ולכן ההבדל בין מכפלה ישרה וסכום ישר יהיה שקוף עבורנו.

מה שכן לא אמרתי עד עכשיו הוא שאני מדבר על מה שנקרא <strong>מכפלה ישרה חיצונית</strong> ו<strong>סכום ישר חיצוני</strong>. במשהו "חיצוני" שכזה אנחנו לוקחים מודולים שלאו דווקא קשורים אחד לשני ו"מדביקים" אותם אחד לשני באופן מלאכותי משהו. אפשר לעשות דבר אחר: מתחילים עם מודול {% equation %}M{% endequation %}, לוקחים <strong>תת-מודולים</strong> שלו, ומסתכלים על מה שקורה כשמחברים אותם. אם התוצאה איזומורפית לסכום הישר החיצוני, הסיטואציה הזו נקראת גם <strong>סכום ישר</strong> ("פנימי", לפעמים). זה מה שלרוב נתקלים בו קודם כשלומדים אלגברה לינארית.

בואו ניזכר איך זה הולך באלגברה לינארית: נניח ש-{% equation %}V{% endequation %} הוא מרחב וקטורי ו-{% equation %}W,U{% endequation %} הם שני תת-מרחבים. אפשר תמיד להסתכל על תת-המרחב {% equation %}W+U\triangleq\left\{ w+u\ |\ w\in W,u\in U\right\} {% endequation %}. אם מתקיימת גם התכונה הנחמדה {% equation %}W\cap U=\left\{ 0\right\} {% endequation %}, אז נותנים לסכום הזה את הסימון המיוחד {% equation %}W\oplus U{% endequation %} וקוראים לו <strong>סכום ישר</strong>. כלומר: "סכום ישר" הוא לא <strong>בניה</strong> אלא הוא <strong>הצהרה</strong> שהבניה של "סכום רגיל של תתי-מרחבים" בנוסף לכך מקיים <strong>משהו</strong>. כמו ש"איחוד זר" זו הצהרה שאיחוד רגיל כלשהו הוא איחוד של שתי קבוצות שהן זרות.

הסיבה שסכום ישר שכזה הוא מעניין היא בגלל תכונת <strong>יחידות</strong> דומה לזו של בסיס. נניח ש-{% equation %}V=W\oplus U{% endequation %}, אז <strong>לכל</strong> {% equation %}v\in V{% endequation %} קיימים איברים <strong>יחידים</strong> {% equation %}w\in W,u\in U{% endequation %} כך ש-{% equation %}v=w+u{% endequation %}. איך יודעים שהוא יחיד? כי נניח ש-{% equation %}w_{1}+u_{1}=v=w_{2}+u_{2}{% endequation %} אז {% equation %}w_{1}-w_{2}=u_{2}-u_{1}{% endequation %}. אגף שמאל הוא ב-{% equation %}W{% endequation %} ואגף ימין הוא ב-{% equation %}U{% endequation %}, ולכן {% equation %}W\cap U=\left\{ 0\right\} {% endequation %} מסיים את הטיעון. אפשר בקלות גם להוכיח ש-{% equation %}W\times U\cong W\oplus U{% endequation %} כאשר {% equation %}W\times U{% endequation %} הוא המכפלה החיצונית של המרחבים; פשוט מגדירים {% equation %}T\left(\left(w,u\right)\right)=w+u{% endequation %} ומקבלים העתקה לינארית שהגרעין שלה הוא ה-{% equation %}\left(w,u\right){% endequation %} שעבורם {% equation %}w=-u{% endequation %} ושוב יש לנו איברים ששייכים ל-{% equation %}W\cap U{% endequation %} ולכן {% equation %}\left(w,u\right)=\left(0,0\right){% endequation %} וההעתקה היא חח"ע (וכמובן שהיא על).

את הרעיון הזה אפשר להכליל לסכום ישר של יותר משני תת-מרחבים; אני אעשה את זה כבר ישירות עבור מודולים, אבל מה שאעשה לא שונה ממה שעושים עבור מרחבים וקטוריים. הסיטואציה שלנו, כאמור, היא של {% equation %}R{% endequation %}-מודול {% equation %}M{% endequation %} עם תתי-מודולים {% equation %}N_{1},\dots,N_{k}{% endequation %}. נניח ש-{% equation %}M=N_{1}+\dots+N_{k}{% endequation %} (לא חייבים להניח את זה; אפשר לתת סימון אחר ל-{% equation %}N_{1}+\dots+N_{k}{% endequation %} במקרה שהוא לא שווה ל-{% equation %}M{% endequation %} ולהמשיך עם הסימון הזה). אני תוהה מה התנאי הנדרש על מנת שיתקיים שכל איבר ב-{% equation %}M{% endequation %} יהיה ניתן לכתיבה <strong>באופן יחיד</strong> בתור {% equation %}n_{1}+\dots+n_{k}{% endequation %} כך ש-{% equation %}n_{i}\in N_{i}{% endequation %}. אבל לפני שנגיע לתנאי הזה, הנה שאלה מכשילה: נניח שזה אכן מה שקורה וכל איבר ב-{% equation %}M{% endequation %} ניתן לכתיבה יחידה שכזו; האם זה אומר ש-{% equation %}M{% endequation %} הוא חופשי מעל {% equation %}N_{1}\cup N_{2}\cup\dots\cup N_{k}{% endequation %}? התשובה היא <strong>ממש לא</strong>. וזה שונה ממה שקורה במרחבים וקטוריים.

מה ההבדל? לפני שאתן דוגמא, דווקא עם ההגדרה הכללית קל לראות אותה לדעתי: <ul>
<li>{% equation %}M{% endequation %} הוא <strong>חופשי</strong> מעל {% equation %}N_{1},\dots,N_{k}{% endequation %} אם לכל {% equation %}m\in M{% endequation %} קיימים ויחידים {% equation %}n_{1},\dots,n_{k}{% endequation %} <strong>וגם</strong> {% equation %}r_{1},\dots,r_{k}\in R{% endequation %} כך ש-{% equation %}m=\sum r_{i}n_{i}{% endequation %}.
</li>
<li>{% equation %}M{% endequation %} הוא <strong>סכום ישר</strong> של {% equation %}N_{1},\dots,N_{k}{% endequation %} אם לכל {% equation %}m\in M{% endequation %} קיימים ויחידים {% equation %}n_{1},\dots,n_{k}{% endequation %} כך ש-{% equation %}m=\sum n_{i}{% endequation %}. </ul> כלומר, מודול חופשי הוא אפילו יותר מאשר סכום ישר - הוא דורש יחידות גם מה<strong>מקדמים</strong>. הנה דוגמא פשוטה לסיטואציה שבה זה לא קורה: {% equation %}R=\mathbb{Z}{% endequation %} ו-{% equation %}M=\mathbb{Z}_{2}\times\mathbb{Z}_{2}{% endequation %}. במקרה הזה {% equation %}M{% endequation %} הוא בוודאי סכום ישר של שני עותקים של {% equation %}\mathbb{Z}_{2}{% endequation %}, אבל {% equation %}\left(1,1\right)=\left(1,0\right)+\left(0,1\right)=3\left(1,0\right)+5\left(0,1\right){% endequation %} מראה שיש יותר מייצוג אחד ל-{% equation %}\left(1,1\right){% endequation %} אם מסתכלים גם על המקדמים. זה מתרחש במקרה הנוכחי פשוט כי {% equation %}3\left(1,0\right)=\left(1,0\right){% endequation %}. בשדה משהו כזה לא יכול לקרות: בגלל שכל איבר שונה מ-0 הוא הפיך, כפל בסקלר הוא העתקה חח"ע - אין שני סקלרים שכפל בהם יחזיר את אותו איבר, ולכן ה"יחידות" באה מעצמה.

נעבור עכשיו לסיכום ביניים של הטענה שאני כן רוצה להוכיח. ראינו עד עכשיו שלוש דרכים להסתכל על עניין ה"סכום ישר פנימי" הזה, בואו נכתוב את כולן.

יש לנו {% equation %}R{% endequation %}-מודול {% equation %}M{% endequation %} עם תת-מודולים {% equation %}N_{1},\dots,N_{k}{% endequation %} כך ש-{% equation %}M=N_{1}+\dots+N_{k}{% endequation %}. אז שלושת התנאים הבאים שקולים: <ul></li>
<li>{% equation %}N_{1}\times\dots\times N_{k}\cong M{% endequation %} עם האיזומורפיזם {% equation %}\varphi\left(n_{1},\dots,n_{k}\right)=n_{1}+\dots+n_{k}{% endequation %}
</li>
<li>לכל {% equation %}1\le i\le k{% endequation %} מתקיים ש-{% equation %}N_{i}\cap\left(N_{1}+\dots+N_{i-1}+N_{i+1}+\dots+N_{k}\right)=\left\{ 0\right\} {% endequation %}
</li>
<li>לכל {% equation %}m\in M{% endequation %} קיימים <strong>ויחידים</strong> {% equation %}n_{1},\dots,n_{k}{% endequation %} כך ש-{% equation %}n_{i}\in N_{i}{% endequation %} ו-{% equation %}m=n_{1}+\dots+n_{k}{% endequation %}
</li>
</ul> שימו לב במיוחד לתנאי האמצעי: שגיאה נפוצה (שגם אני עשיתי) היא להניח שההכללה של ה-{% equation %}W\cap U=\left\{ 0\right\} {% endequation %} מהמקרה של סכום ישר של שני מרחבים היא פשוט להגיד שצריך להתקיים {% equation %}N_{i}\cap N_{j}=\left\{ 0\right\} {% endequation %} לכל {% equation %}i\ne j{% endequation %}. אבל הדרישה הזו חלשה מדי ולא מספיקה (תרגיל נחמד: למצוא דוגמא נגדית) ולכן אני משתמש בדרישה הכללית יותר, שלכל תת-מודול {% equation %}N_{i}{% endequation %}, לא יהיה לו שום דבר לא טריוויאלי משותף עם מה שמתקבל מכל המרחבים האחרים ביחד.

הדרך האהובה ביותר להוכיח שכמה טענות הן שקולות היא "שרשרת גרירות" - נוכיח שהטענה הראשונה גוררת את השניה, השניה את השלישית, השלישית את הראשונה - וזהו, זה מוכיח שהכל גורר את הכל.

בשביל להוכיח את הטענה השניה בעזרת הראשונה, ניקח {% equation %}a\in N_{i}\cap\left(N_{1}+\dots+N_{i-1}+N_{i+1}+\dots+N_{k}\right){% endequation %} ונוכיח ש-{% equation %}a=0{% endequation %}. אנחנו יודעים שמצד אחד {% equation %}a\in N_{i}{% endequation %} ומצד שני {% equation %}a=\sum_{j\ne i}a_{j}{% endequation %} עבור {% equation %}a_{j}\in N_{j}{% endequation %}. מכאן ש-{% equation %}\varphi\left(a_{1},a_{2},\dots,a_{i-1},-a,a_{i+1},\dots,a_{k}\right)=\sum_{j\ne i}a_{j}-a=0{% endequation %} ולכן {% equation %}\left(a_{1},\dots,a,\dots,a_{k}\right)\in\ker\varphi{% endequation %}. מכיוון ש-{% equation %}\varphi{% endequation %} איזומורפיזם ובפרט חח"ע, נקבל {% equation %}\left(a_{1},\dots,a,\dots,a_{k}\right)=\left(0,0,\dots,0\right){% endequation %} ובפרט {% equation %}a=0{% endequation %}.

בשביל להוכיח את הטענה השלישית בעזרת השניה, נשים לב ראשית כל לכך שהנחנו מראש ש-{% equation %}M=N_{1}+\dots+N_{k}{% endequation %} וזה מראה מייד את ה"קיימים", האתגר הוא רק להראות את ה"יחידים". זו בעצם אותה הוכחה שכבר הראיתי קודם: נניח ש-{% equation %}m=a_{1}+\dots+a_{k}=b_{1}+\dots+b_{k}{% endequation %} כך ש-{% equation %}a_{i},b_{i}\in N_{i}{% endequation %}, אז {% equation %}\left(a_{1}-b_{1}\right)+\dots+\left(a_{k}-b_{k}\right)=0{% endequation %} ולכן, לכל {% equation %}i{% endequation %}, נקבל {% equation %}-\left(a_{i}-b_{i}\right)=\sum_{j\ne i}\left(a_{j}-b_{j}\right){% endequation %}. אגף שמאל שייך ל-{% equation %}N_{i}{% endequation %} ואגף ימין ל-{% equation %}\sum_{j\ne i}N_{j}{% endequation %} ולכן המסקנה מהטענה השניה היא ש-{% equation %}a_{i}-b_{i}=0{% endequation %}, כלומר {% equation %}a_{i}=b_{i}{% endequation %}.

בשביל להוכיח את הטענה הראשונה בעזרת השלישית משתמשים בעוד טיעון שכבר פחות או יותר הראיתי. זה ש-{% equation %}\varphi{% endequation %} הוא הומומורפיזם זה קל לבדיקה; זה שהוא על {% equation %}M{% endequation %} נובע מ-{% equation %}M=N_{1}+\dots+N_{k}{% endequation %}; נשאר רק לראות ש-{% equation %}\varphi{% endequation %} חח"ע, כלומר ש-{% equation %}\ker\varphi=\left\{ \left(0,0,\dots,0\right)\right\} {% endequation %}. מכיוון ש-{% equation %}0\in M{% endequation %} ו-{% equation %}0=0+0+\dots+0{% endequation %} כאשר {% equation %}0\in N_{i}{% endequation %} לכל {% equation %}1\le i\le k{% endequation %}, זו ההצגה <strong>היחידה</strong> של 0 בתור {% equation %}n_{1}+\dots+n_{k}{% endequation %}, ולכן האיבר היחיד בגרעין של {% equation %}\varphi{% endequation %} הוא {% equation %}\left(0,0,\dots,0\right){% endequation %}. זה מסיים את ההוכחה.

יש עוד דבר אחד שאני רוצה "להציל" מאלגברה לינארית, והוא ידרוש ממני להתחיל להתכנס אל היעד שלנו. כזכור, אני רוצה בסופו של דבר להוכיח משפט עבור מודולים <strong>נוצרים סופית</strong> מעל <strong>תחום ראשי</strong>. "תחום ראשי" הוא תחום שלמות (חוג <strong>קומוטטיבי</strong> ללא מחלקי אפס) שבנוסף לכך כל אידאל בו נוצר על ידי איבר בודד; אז במסגרת ההתכנסות, אני אניח כרגע ש-{% equation %}R{% endequation %} הוא תחום שלמות, ואני אדבר על מודול <strong>חופשי</strong> שהוא נוצר סופית. כלומר, {% equation %}M{% endequation %} יהיה {% equation %}R{% endequation %}-מודול חופשי מעל קבוצת יוצרים סופית {% equation %}A{% endequation %}. ניתן לזה שם: {% equation %}\text{rank}M=\left|A\right|{% endequation %} - <strong>הדרגה</strong> של המודול החופשי {% equation %}M{% endequation %}. זו המקבילה ל<strong>מימד</strong> עבור מרחבים וקטוריים. מה שאני רוצה להציל כרגע הוא משפט שימושי מאלגברה לינארית: אם {% equation %}V{% endequation %} מרחב וקטורי מעל {% equation %}F{% endequation %} ו-{% equation %}\dim V=n{% endequation %}, אז הקבוצה {% equation %}\left\{ v_{1},\dots,v_{n+1}\right\} {% endequation %} של {% equation %}n+1{% endequation %} וקטורים <strong>שונים</strong> מעל {% equation %}V{% endequation %} היא <strong>תלויה לינארית</strong>, כלומר קיימים {% equation %}\lambda_{1},\dots,\lambda_{n+1}\in F{% endequation %} שלא כולם אפס כך ש-{% equation %}\sum\lambda_{i}v_{i}=0{% endequation %}. עכשיו תחליפו את {% equation %}V{% endequation %} ב-{% equation %}M{% endequation %} ואת {% equation %}\dim V{% endequation %} ב-{% equation %}\text{rank}M{% endequation %} וקחו את הסקלרים מתוך {% equation %}R{% endequation %} וקיבלתם את הטענה שאני רוצה להוכיח.

האם אתם זוכרים איך מוכיחים את הטענה הזו באלגברה לינארית? כמו רוב הדברים באלגברה לינארית בסיסית היא ניתנת לרדוקציה זריזה אל משהו שמדבר על מטריצות, ואותו משהו הוא זוועת עולם טכנית. מה שעושים הוא להסתכל על בסיס {% equation %}B=\left\{ e_{1},\dots,e_{n}\right\} {% endequation %} של {% equation %}V{% endequation %} ולכתוב את ה-{% equation %}v_{i}{% endequation %}-ים באמצעותו: {% equation %}v_{i}=\sum_{j=1}^{n}a_{j,i}e_{j}{% endequation %}. זה מגדיר לנו מטריצה {% equation %}A=\left(a_{j,i}\right){% endequation %} עם {% equation %}n{% endequation %} שורות ו-{% equation %}n+1{% endequation %} עמודות, כך שהעמודה ה-{% equation %}i{% endequation %} הוא וקטור הקואורדינטות של {% equation %}v_{i}{% endequation %} על פי הבסיס {% equation %}B{% endequation %}.

כעת, נסתכל על צירוף לינארי של ה-{% equation %}v_{i}{% endequation %}-ים שאיבריו הם <strong>משתנים</strong> ואנחנו רוצים שיהיה שווה ל-0: {% equation %}\sum_{i=1}^{n+1}x_{i}v_{i}=0{% endequation %}. אם נפתח את ההגדרה של כל {% equation %}v_{i}{% endequation %}, נקבל את המשוואה:

{% equation %}\sum_{i=1}^{n+1}x_{i}\sum_{j=1}^{n}a_{j,i}e_{j}=0{% endequation %}

אפשר לשנות את סדר הסכימה ולקבץ איברים לפי ה-{% equation %}e_{j}{% endequation %}-ים:

{% equation %}\sum_{j=1}^{n}\left(\sum_{i=1}^{n+1}a_{j,i}x_{i}\right)e_{j}=0{% endequation %}

מכיוון ש-{% equation %}B{% endequation %} הוא קבוצה בלתי תלויה לינארית, נסיק שהמקדמים של כל {% equation %}e_{j}{% endequation %} הם 0:

{% equation %}\sum_{i=1}^{n+1}a_{j,i}x_{i}=0{% endequation %} לכל {% equation %}1\le j\le n{% endequation %}

אם נסמן {% equation %}\overline{x}=\left[\begin{array}{c} x_{1}\\ \vdots\\ x_{n+1} \end{array}\right]{% endequation %} אז נקבל את המשוואה

{% equation %}A\overline{x}=\overline{0}{% endequation %}

זו מערכת משוואות עם יותר נעלמים ({% equation %}n+1{% endequation %}) מאשר משוואות ({% equation %}n{% endequation %}) ולכן קיים לה פתרון לא טריוויאלי. דרך אחרת לחשוב על זה: נרחיב את {% equation %}A{% endequation %} למטריצה מסדר {% equation %}\left(n+1\right)\times\left(n+1\right){% endequation %} על ידי הוספת שורת אפסים. גם {% equation %}A{% endequation %} המורחבת תקיים את המשוואה {% equation %}A\overline{x}=\overline{0}{% endequation %}, ומכיוון שיש לה שורת אפסים אז {% equation %}\det A=0{% endequation %} ולכן {% equation %}A{% endequation %} לא הפיכה ולכן הגרעין של ההעתקה {% equation %}A\overline{x}{% endequation %} אינו טריוויאלי. איך שאני לא רוצה לנסח את זה, אתם רואים שאני נעצר מתישהו בהוכחה הזו בדיוק לפני השלב הטכני המזוויע שאני מדבר עליו.

ולמה טרחתי להזכיר מה קורה באלגברה לינארית, הרי אנחנו לא באלגברה לינארית יותר? ובכן, להוכיח את המשפט עבור מודולים במקרה שלנו הוא בסך הכל <strong>רדוקציה</strong> למקרה של מרחבים וקטוריים. זה היתרון הגדול שבעבודה עם <strong>תחום שלמות</strong> {% equation %}R{% endequation %}; כבר ראינו <a href="https://gadial.net/2018/01/30/rings_of_fractions/">בסדרת הפוסטים</a> הזו שכל תחום שלמות {% equation %}R{% endequation %} ניתן לשיכון בשדה {% equation %}F{% endequation %} ("שדה השברים" של {% equation %}R{% endequation %}). כעת, מכיוון ש-{% equation %}M{% endequation %} הוא {% equation %}R{% endequation %}-מודול חופשי מדרגה {% equation %}n{% endequation %} אז {% equation %}M\cong R^{n}{% endequation %}, ולכן {% equation %}M\subseteq F^{n}{% endequation %} (ההכלה הזו היא דרך מקוצרת לומר "{% equation %}M{% endequation %} איזומורפי לתת-מרחב של {% equation %}F^{n}{% endequation %}"). מכיוון ש-{% equation %}F{% endequation %} הוא שדה אז {% equation %}F^{n}{% endequation %} הוא מרחב וקטורי, כך שאם {% equation %}\left\{ a_{1},\dots,a_{n+1}\right\} {% endequation %} הם {% equation %}n+1{% endequation %} איברים שונים של {% equation %}M{% endequation %}, אנחנו יודעים שקיימים {% equation %}\lambda_{1},\dots,\lambda_{n+1}\in F{% endequation %} כך ש-{% equation %}\sum\lambda_{i}a_{i}=0{% endequation %}. האם סיימנו?

עדיין לא לגמרי סיימנו, כי אותם {% equation %}\lambda_{i}\in F{% endequation %} הם לא בהכרח איברים של {% equation %}R{% endequation %}. כאן נכנסת לתמונה באופן חזק העובדה ש-{% equation %}F{% endequation %} הוא שדה השברים של {% equation %}R{% endequation %}; מתקיים {% equation %}\lambda_{i}=\frac{r_{i}}{s_{i}}{% endequation %} עבור {% equation %}r_{i},s_{i}\in R{% endequation %}. אם נגדיר {% equation %}s=\prod s_{i}{% endequation %}, אז {% equation %}s\lambda_{i}\in R{% endequation %} לכל {% equation %}\lambda_{i}{% endequation %}, כך שנקבל {% equation %}\sum\left(s\lambda_{i}\right)a_{i}=0{% endequation %} ב-{% equation %}M{% endequation %} כאשר כאן המקדמים הם איברים של {% equation %}R{% endequation %} - וזה מה שרצינו.

זהו, הצלחנו להציל את כל האלגברה הלינארית שנזדקק לה בהמשך; השלב הבא הוא לדבר קצת על מה זה אומר שמודול הוא <strong>נוצר סופית</strong> ואחר כך נגיע הישר אל המשפט המרכזי שאנחנו רוצים להוכיח.
